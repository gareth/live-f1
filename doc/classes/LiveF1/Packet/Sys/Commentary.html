Class LiveF1::Packet::Sys::Commentary
In: lib/live-f1/packet.rb
Parent: Sys

Commentary packets contain the commentary provided by the live timing stream.

Each packet contains a maximum of 125 bytes of commentary. A commentary string longer than this will be transmitted in successive Commentary packets with consecutive line numbers
